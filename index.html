Soil Health Monitoring System

About The Project

This project is designed to monitor soil health in real-time using IoT sensors. It detects NPK levels, moisture, temperature, and alerts nearby farmers if pesticides are detected. Blockchain technology ensures that the data is secured and tamper-proof.
